Ruby Net::SMB - SMB/CIFS client class for Ruby
======================================================================

Synopsis
----------------------------------------------------------------------

This is a Samba libsmbclient binding, i.e., SMB/CIFS client class
for Ruby.

Requirement
----------------------------------------------------------------------

For runtime environment:

  * Ruby 1.9.3+
  * Samba 3.6+ (libsmbclient)

For build and development environment:

  * C compiler
  * Ruby bundler
  * Ruby rake-compiler
