Ruby Net::SMB - SMB/CIFS client class for Ruby
======================================================================

Synopsis
----------------------------------------------------------------------

This is a Samba libsmbclient binding, i.e., SMB/CIFS client class
for Ruby.

Requirement
----------------------------------------------------------------------

* Ruby 1.9.3+
* Samba 3.6+ (libsmbclient)
* C compiler
